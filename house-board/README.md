# The House Board

Статичное PWA-приложение для учёта дежурств по дому: чек-листы уборки по зонам (ванная, кухня, общие зоны), ротация дежурств, «карма» жильцов, список расходников и подсказки по бытовым приборам. Данные хранятся в `localStorage` браузера, бэкенд не требуется.

## Запуск

Это статические файлы — им нужен любой HTTP-сервер (открытие `index.html` напрямую по `file://` не даст работать `manifest.json` и service worker/PWA-функциям):

```bash
cd house-board
python3 -m http.server 8000
```

Открыть http://localhost:8000 в браузере.

## Структура

- `index.html` — вся разметка, стили и логика приложения (без сборки).
- `manifest.json` — манифест PWA для установки на главный экран.
- `icon-192.png`, `icon-512.png`, `apple-touch-icon-180.png` — иконки приложения.
- `assets/` — фото чистящих средств и звуки уведомлений.
